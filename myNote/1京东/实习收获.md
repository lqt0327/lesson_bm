## 目录结构

```
src
├── client
│   ├── components
│   │   ├── Shared ----组件库
│   │   ├── Feature
│   │   ├── Page ----页面
│   │   └── Scene ----应用场景
│   ├── reducers
│   ├── sagas
│   ├── selectors
│   ├── services ----接口方法
│   ├── constants ----常量
│   ├── utils ----工具方法
│   ├── styles ----全局样式
│   │   ├── mixins.styl
│   │   └── variables.styl
│   ├── routes ----应用路由
│   ├── main.jsx
│   └── store.js
└── server ----本地服务器
    ├── middlewares ----中间件
    ├── routers 
    ├── mocks ----模拟数据源
    ├── app.js
    └── index.js
```
