## 上面的json 通过 json-schema 打印输出 ( traverse(schema3, {allKeys: true, cb: {pre, post}}); // 需要开启 allKeys: true，不然就必须用properties: {}在最外面套一层)
```

  }
} ---分割线--- /banner1

  title: 'title'
} ---分割线--- /carousel2/carousel

```
